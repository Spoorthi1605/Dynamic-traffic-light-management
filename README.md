Urban traffic congestion is still the most significant issue for Transportation Systems. Classical traffic signal control strategies usually do not adjust well to quickly changing vehicle densities. The goal of this project is to create a fully adaptive, decentralized Dynamic Traffic Light Management System intended to reduce traffic congestion, decrease vehicle waiting times, shorten queue lengths, and maximize traffic flow in general. The suggested method utilizes a Multi-Agent Deep Reinforcement Learning (MARL) framework, where every traffic intersection is an independent agent that is able to learn and make decisions on its own. Each agent employs Long Short-Term Memory (LSTM) to learn temporal dependencies in traffic patterns and forecast the movement pattern of vehicles over time. In order to improve spatial awareness, Convolutional Block Attention Modules (CBAM) are used within the network so that agents can pay attention to key traffic areas at each intersection. Furthermore, a Dueling Double Deep Q-Network (D3QN) architecture is used to guarantee stable learning and accurate estimation of optimal actions to control traffic signals, facilitating agents to make the most effective phase-switching decisions based on different traffic conditions. The most important performance measures witnessed are the Average Waiting Time (AWT) of 0.35 seconds, Average Turnaround Time (ATT) of 72.93 seconds, and Average Queue Length (AQL) of 5 vehicles. Looking ahead, the project envisions incorporating real-time traffic data, adaptability to live traffic conditions, various weather scenarios and unforeseen traffic incidents to ensure robustness and reliability.
